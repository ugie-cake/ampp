# Industry Experience AMPP Repository

This repository contains the following items:

- **XAMPP for Windows (IE Distro)**: A version of XAMPP simplified and updated for use in IE units

The latest release can be found [here](https://github.com/ugie-cake/ampp/releases/latest). To learn more about this distro, read [here](https://github.com/ugie-cake/ampp/blob/main/README.xampp.md). 

- **AMPP for macOS setup script**: A script that installs and configures a PHP development environment with Homebrew.

This script can be found [here](https://github.com/ugie-cake/ampp/blob/main/macos_ampp.sh). You can run the script directly in your terminal by pasting the following command in your macOS terminal: 

`/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/ugie-cake/ampp/HEAD/macos_ampp.sh)"`
